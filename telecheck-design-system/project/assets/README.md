## Icons

### System icons
We use **[Lucide](https://lucide.dev)** via CDN:
```html
<script src="https://unpkg.com/lucide@latest"></script>
<script>lucide.createIcons();</script>
```
Or per-icon SVG import from unpkg.

Default size: 20 px. Stroke: 1.5. Linecap/linejoin: round.

### Custom icons (brand-specific)
- `icons/ai-spark.svg` — the Telecheck AI attribution glyph (§12, §16). Used anywhere AI-authored content appears, in Iris (`--iris-500`).
